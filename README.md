# plandrop

Push a static HTML document to a unique, secure hostname on your network and share the link — the hosted assets are served with zero server-side logic.

```
npx plandrop create        # mint a host (hostname + passphrase) into a repo-root dotfile
npx plandrop <command>     # manage the hosting over an authed, WebDAV-like surface
```

**Status:** early development.
